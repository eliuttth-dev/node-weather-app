# WEATHER CLI-TOOL

This is a basic weather cli tool that shows you weather information inside the terminal

## Installation

1. Clone this repo

2, Install dependencies
`npm install`

3. Create `.env` file
`RAPIDAPI_KEY=your_api_key`

4. Link the CLI tool
`npm link`

5. Run it as a CLI tool
`weather-cli --city <city-name>`
